| Operation       | Linked List      | ArrayList            |
|-----------------|------------------|----------------------|
| Append          | 👍 O(1)          |  👍 O(1)            |
| Remove Last     | 🔴 O(n)          |  🔵 O(1)            |
| Prepend         | 🔵 O(1)          |  🔴 O(n)            |
| Remove First    | 🔵 O(1)          |  🔴 O(n)            |
| Insert          | 👍 O(n)          |  👍 O(n)            |
| Remove          | 👍 O(n)          |  👍 O(n)            |
| Lookup by Index | 🔴 O(n)          |  🔴 O(1)            |
| Lookup by Value | 👍 O(n)          |  👍 O(n)            |

* Note: 
  + 🔴 indicates bad performance, 🔵 indicates good performance, 👍 indicates both works with same performance.
